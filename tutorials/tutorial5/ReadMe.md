# Tutorial 5 - Control Flow: The if-else and switch statements.



## if statement


```cpp
int x = 10;
int y = 20;
if (x > y) {
  printf("x is greater than y");
}
```


### if-else statement

```cpp
int x = 10;
int y = 20;
if (x > y) {
  printf("x is greater than y");
}
else
{
    printf("x is greater than y");
}
```
